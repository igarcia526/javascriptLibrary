WHAT HELPS ME LEARN CODE?

YOUTUBE

Watching videos on youtube helps me grasp difficult conscepts much easier than sitting in a lecture. Short detailed videos help me stay focused on the subject.

OTHER'S CODE

Looking at other people's code is a great way to see how code works, and helps me develop good coding habits. Looking at others' code is a great way to learn new things!

WHAT DO I STRUGGLE WITH THE MOST?

REMEMBERING COMMANDS

HTML, CSS, and JavaScript all have their own keywords and tags, sometimes remembering all those words and what they are used for can be a bit overwhelming, thankfully we have GOOGLE!

PUTTING IT TO USE

Its a great feeling when you finally understand a subject youve been struggling with, yet this awesome feeling can sometimes be overshadowed when you cant think of a way to use your new-found knowledge. This is why looking at other people's code can be helpful!
